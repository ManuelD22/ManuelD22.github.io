	<h2>Que significa HTML</h2>
	<p>
	    HyperText Markup Language o Lenguaje de Marcado de Hipertexto, es un lenguaje utilizado para definir la estructura y el contenido de las páginas web.

	<h2 id="html">Etiqueta html</h2>
	<p>Indica la lectura del archivo en html</p>
	<h2>Etiqueta head</h2>
	<p>Se declaran las caracterisricas que tendra el html</p>
	<h2>Etiqueta body</h2>
	<p>Se integra todo el contenido que incluira el html</p>
	<h2>Etiquetas h</h2>
	<p>Se utilizan para definir un titulo (h1) y varios subtitulos dentro del html</p>
	<h2>Etiqueta br</h2>
	<p>Se utiliza para que el texto vaya al siguiente renglon</p>
	<h2>Etiquetas em, i, strong y b</h2>
	<p>Las etiquetas em o i se utilizan para aplicar el <em>tipo de letra</em> <i>italica</i>
	<br>
	Mientras que las etiquetas strong o b se utilizan para aplicar el <strong>tipo de letra</strong> <b>negritas</b></p>
	<h2 id="listas">Etiqueta ul y ol</h2>
	<p>La etiqueta ul se utiliza para elaborar listas desordenadas</p>
	<br>
	<p>La etiqueta ol se utiliza para elaborar listas numeradas</p>

Lenguaje HTML

Que significa HTML

HyperText Markup Language o Lenguaje de Marcado de Hipertexto, es un lenguaje utilizado para definir la estructura y el contenido de las páginas web.
Utiliza etiquetas para organizar y presentar elementos en un navegador.
Se complementa con CSS para el diseño y con JavaScript para la interactividad

Etiquetas

Las etiquetas que vamos a usar son

  • HTML
  • head
  • body
  • h1,2,3,4,5,6
  • br
  • em, strong
  • ul, ol

Etiqueta html

Indica la lectura del archivo en html

Etiqueta head

Se declaran las caracterisricas que tendra el html

Etiqueta body

Se integra todo el contenido que incluira el html

Etiquetas h

Se utilizan para definir un titulo (h1) y varios subtitulos dentro del html

Etiqueta br

Se utiliza para que el texto vaya al siguiente renglon

Etiquetas em, i, strong y b

Las etiquetas em o i se utilizan para aplicar el tipo de letra italica
Mientras que las etiquetas strong o b se utilizan para aplicar el tipo de letra negritas

Etiqueta ul y ol

La etiqueta ul se utiliza para elaborar listas desordenadas


La etiqueta ol se utiliza para elaborar listas numeradas
